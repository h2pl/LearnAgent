## 分类逻辑：按能力分，不按厂商分

多模态产品的分类有很多种砍法——可以按厂商（OpenAI/Google/Anthropic）、可以按闭源开源、可以按免费付费。但这些分类对你没有直接意义——你关心的不是"谁家的"，而是"能不能解决我的问题"。

本文用 **"你能用它们做什么"** 作为分类标准，把 2026 年的多模态产品分成 6 大品类。

## 品类一：全能对话

**在一个聊天界面里，文字+图片+语音+生图全都能干。** 这是目前用户量最大、竞争最激烈的品类。

| 产品 | 核心能力 | 一句话定位 |
|------|---------|-----------|
| **ChatGPT（GPT-4o）** | 聊天+看图+语音对话+DALL-E 生图+代码解释 | 月活 200 亿，最全能的多模态助手 |
| **Gemini** | 文字+图片+视频理解+语音，原生多模态 | 视频理解最强，Google 全家桶深度整合 |
| **豆包（字节跳动）** | 聊天+看图+生图+生视频+语音，国内生态 | 国内月活 3.15 亿第一，中文体验最佳 |

**一句话判断**：如果你只打算用一个产品覆盖所有多模态需求，从这三家里选一个就够了。

## 品类二：图像生成

**输入文字描述，输出高质量图片。** 从海报到插画到产品图，一句话搞定。

| 产品 | 核心能力 | 一句话定位 |
|------|---------|-----------|
| **Midjourney** | 文字生图，审美天花板 | 图像质量公认最高，设计师首选 |
| **DALL-E（OpenAI）** | ChatGPT 内置生图，文字理解强 | Prompt 跟随度最佳，长文字也能准确渲染 |
| **豆包 Seedream** | 字节系，中文 Prompt 理解好 | 国产首选，电商/自媒体场景优势明显 |
| **Stable Diffusion / Flux** | 开源路线，可本地部署，高度可控 | 需要定制化、本地私密运行的场景 |

## 品类三：视频生成

**输入文字或图片，输出视频片段。** 2026 年视频生成迎来拐点——Sora 官宣停运，Runway 和国产方案崛起。

| 产品 | 核心能力 | 一句话定位 |
|------|---------|-----------|
| **Runway Gen-4** | 文字/图片生视频，专业创作工具 | 视频质量最高，海外创意从业者首选 |
| **可灵 AI** | 文字/图片生视频，中文优先 | 国产视频生成的标杆，中文 Prompt 体验最好 |
| **Seedance（字节跳动）** | 文生视频+图生视频，豆包生态 | 与豆包联动，一键从对话到视频 |
| **Google Veo** | Gemini 体系内的视频生成 | 与 Gemini 打通，但目前仅限部分用户 |

> Sora 已于 2026 年 4 月关闭消费者端服务，API 将于 9 月停用。视频生成的主力已转至 Runway 和国产方案。

## 品类四：语音交互

**让 AI 能听会说。** 从录音转文字到实时语音对话到语音克隆，应用场景极广。

| 产品 | 核心能力 | 一句话定位 |
|------|---------|-----------|
| **OpenAI Realtime API** | 实时语音对话，WebSocket 流式 | 开发者集成首选，延迟最低 |
| **ElevenLabs** | 语音合成+语音克隆，情感控制 | 语音质量天花板，多语言支持 |
| **Whisper（OpenAI）** | 开源语音识别，本地可部署 | 最好的开源 STT，支持 99 种语言 |
| **通义听悟（阿里）** | 中文语音识别，实时转写+纪要 | 国内会议记录场景首选 |

## 品类五：音乐音频

**一句话生成完整歌曲，或一段背景音乐。** 这个品类刚起步，但产品体验已经足够惊人。

| 产品 | 核心能力 | 一句话定位 |
|------|---------|-----------|
| **Suno** | 文字→完整歌曲（作词+编曲+演唱） | 现象级产品，一句话出歌 |
| **Udio** | 文字→音乐，声音质感更精细 | Suno 的主要竞争对手，人声更自然 |

## 品类六：开发者 API

**不通过聊天界面，直接在代码里调用多模态能力。** 如果你在开发一个需要 AI 能力的应用，这是你最关心的品类。

| 产品 | 核心能力 | 使用方式 |
|------|---------|---------|
| **OpenAI Vision API** | 图片理解、OCR、图表分析 | REST API，gpt-4o 模型 |
| **Claude Vision API** | 文档理解、PDF 解析、UI 分析 | REST API，逻辑推理强 |
| **Gemini Vision API** | 图片+视频理解，长上下文 | REST API，视频理解领先 |
| **HuggingFace** | 开源多模态模型（LLaVA/Qwen-VL/InternVL） | 本地部署或 API 调用 |

## 如何选：一张决策表

| 你的需求 | 最佳选择 | 备选 |
|----------|---------|------|
| 日常聊天+偶尔看图+偶尔生图 | ChatGPT | Gemini、豆包 |
| 生成高质量图片 | Midjourney | DALL-E |
| 中文生态，图文音视频全链路 | 豆包 | — |
| 生成视频 | 可灵（中文）/ Runway（海外） | Seedance、Veo |
| 语音克隆或高精度 TTS | ElevenLabs | OpenAI TTS |
| 生成一首歌 | Suno | Udio |
| 在代码里调用视觉能力 | OpenAI Vision API | Gemini Vision API |
| 本地部署多模态模型 | HuggingFace + Ollama | — |

## 总结

- 多模态产品按能力分成 **6 大品类**：全能对话、图像生成、视频生成、语音交互、音乐音频、开发者 API
- **全能对话品类竞争最激烈**：ChatGPT（200 亿月活）、Gemini（7.5 亿月活）、豆包（3.15 亿月活）三足鼎立
- **视频生成进入新阶段**：Sora 退出，Runway 和可灵成为新标杆
