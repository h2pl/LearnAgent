# 06 — 多模态工程：评估、成本与生产交付

从 demo 到生产，多模态 AI 需要补足工程化的最后一公里。本章覆盖评估指标体系的建立、多模态特有的成本模型与优化策略、可观测性方案、安全治理，以及一份完整的上线 checklist。

## 学习路径

本章从评估到交付，按"度量→优化→监控→治理→上线"的顺序展开。

### 评估指标体系

多模态评估比纯文本复杂得多——每个模态有自己的评估指标。理解侧用准确率/召回率/BLEU，生成侧用 FID/CLIP Score/人工评估，综合任务还需要端到端指标。

### 成本建模与优化

多模态的 token 消耗是文本的数十到数百倍。图像 token 怎么算？视频帧如何处理？缓存策略怎么设计？本章给出完整的成本分析框架。

### 可观测性与调试

多模态链路长、故障点多。从 tracing 到 logging，从输入审计到输出验证，建立全链路可观测性是多模态应用上线的必要条件。

### 安全治理

多模态引入了新的攻击面：提示注入通过图像传播、生成内容合规、隐私泄露。本章覆盖 OWASP 多模态安全最佳实践。

### 上线 checklist

一份从模型选型到灰度发布的完整 checklist，确保上线前不漏掉任何关键环节。
