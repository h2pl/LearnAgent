# 01 — 多模态基础：认知与机制

在深入多模态的具体能力之前，先建立三层基础认知：多模态 AI 的全景版图、模型内部的工作机制、以及跨模态表示学习的核心原理。

## 学习路径

本章是进阶栏目的起点，三篇文章构成"全景→机制→表示"的递进关系。

### 1. 建立全景认知

[多模态 AI 全景](./01-multimodal-landscape.md) 回答最根本的问题：多模态是什么、为什么需要、2026 年的模型各支持哪些模态。读完你会对多模态的技术版图有清晰的判断。

### 2. 理解底层机制

[多模态核心机制](./02-core-mechanisms.md) 进入模型内部，讲清楚一个关键区分——多模态 AI 其实是两条完全不同的技术路线（理解侧用 Transformer，生成侧用扩散模型），然后深入理解侧的三个核心机制：编码、融合、对齐。

### 3. 跨模态表示学习

[跨模态表示学习](./03-cross-modal-representation.md) 讲清楚 CLIP 等模型如何让图像和文字在同一个向量空间中"对话"。这是多模态 RAG、图像搜索、zero-shot 分类等应用的底层基础。
