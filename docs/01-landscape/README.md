# 01 — 生态认知

Agent 生态全景认知。本章从「什么是 Agent」这个起点出发，建立对 AI Agent 领域的整体认知框架。

先通过 [5W2H 框架](./01-what-is-agent.md) 搞清楚 Agent 的本质定义、核心机制和适用场景，再用 [七大品类 × 五级能力](./02-mainstream-agents.md) 的双维框架看清市场全貌。最后通过三个明星产品——[Claude Code](./03-claude-code.md)（编程 Agent 标杆）、[OpenClaw](./04-openclaw.md)（本地优先个人助理）、[Hermes Agent](./05-hermes-agent.md)（自主成长开源 Agent）——理解不同品类的设计哲学。
