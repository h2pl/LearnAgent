# 09 — 记忆管理

RAG 帮你从外部知识库检索信息。但 Agent 还需要记住**自己的经历**——上次对话做了什么决策、用户偏好是什么、在哪出过错。**记忆管理让 Agent 从"金鱼记忆"变成"老搭档"**。
